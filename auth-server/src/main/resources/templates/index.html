OAuth 2.0 Authorization Server

Authorization Server Dashboard

This is the OAuth 2.0 Authorization Server with PKCE support.

Available Endpoints:
  • Authorization: http://localhost:9000/auth/oauth2/authorize
  • Token: http://localhost:9000/auth/oauth2/token
  • JWK Set: http://localhost:9000/auth/oauth2/jwks
  • User Info: http://localhost:9000/auth/userinfo
Registered Clients:
  • Web App: web-app (PKCE enabled)
  • Mobile App: mobile-app (PKCE enabled)
  • Test Client: test-app (PKCE disabled for testing)
